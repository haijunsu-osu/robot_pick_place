Step-by-Step: Install ROS 2 Humble, MoveIt, and Universal Robots on Windows (WSL)

1. Install WSL and Ubuntu 22.04

  • Enable Windows Subsystem for Linux (WSL) and install Ubuntu 22.04 from the Microsoft Store.
  • wsl --install -d Ubuntu-22.04
  • Restart your computer if prompted. Open Ubuntu from the Start menu.
  • Test WSL installation:
  • lsb_release -a

2. Install ROS 2 Humble

  • Follow the official ROS 2 Humble installation guide for Ubuntu 22.04: ROS 2 Humble Install
  • Major steps:
  • sudo apt update
  • sudo apt install ros-humble-desktop
  • Test ROS 2 installation:
  • ros2 run demo_nodes_cpp talker

3. Install MoveIt for ROS 2 Humble

  • Install MoveIt using apt:
  • sudo apt install ros-humble-moveit
  • Test MoveIt installation (launch RViz demo):
  • ros2 launch moveit2_tutorials demo.launch.py

4. Install Universal Robots ROS2 Driver

  • Install the Universal Robots package:
  • sudo apt install ros-humble-ur
  • Test UR package installation (list UR launch files):
  • ros2 pkg list | grep ur

5. Run UR5e Step Demo in RViz2

  • Download the demo script ur5e_step_20deg.py from the course repository:
  • ur5e_step_20deg.py
  • Make sure ROS 2 and MoveIt are sourced:
  • source /opt/ros/humble/setup.bash
  • Run the script to move the UR5e robot in RViz2 (each joint moves 20 degrees):
  • python3 ur5e_step_20deg.py
  • Follow the instructions for environment setup and verification
