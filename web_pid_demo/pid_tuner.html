PID Control Demo: Electric Servo Motor

This interactive demo explains the concept of PID (Proportional-Integral-Derivative) control using an electric servo motor as an example. Adjust the PID parameters and observe how the motor's position responds to a step change in the target angle.

Blue: Simulated | Green: Real Motor | Red: Target

How PID Control Works

  • Proportional (P): Reacts to the current error (difference between target and actual position).
  • Integral (I): Reacts to the accumulation of past errors (helps eliminate steady-state error).
  • Derivative (D): Reacts to the rate of change of the error (helps dampen oscillations).

Try changing the P, I, and D values to see how the servo motor's response changes!
